# 단어 도감: Word Quest — 게임 기획서 v1

> "단어 시험 범위가 던전이 되고, 틀린 단어는 보스가 된다."
> 간격 반복(Spaced Repetition) 복습을 16비트 JRPG 턴제 전투로 포장한 영단어 학습 게임.

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 타깃 | 고1 (내신·수능 영어 단어) |
| 형태 | 단일 HTML 파일 웹앱 (브라우저에서 열면 동작, 설치 불필요) |
| 톤 | 정통 판타지 + 유머 한 숟가락 ("야생의 ABANDON몬이 나타났다!") |
| 저장 | 브라우저 localStorage + 백업 내보내기/가져오기 |
| 학습 원리 | 간격 반복 + 능동 회상 (에빙하우스 망각곡선 기반) |

## 2. 코어 루프

1. **영입** — 시험 범위 단어를 복붙 등록 → 몬스터가 던전에 서식
2. **입장** — 매일 던전에 입장, 오늘 출현하는 몬스터와 연속 전투
3. **전투** — 뜻 4지선다. 정답 = 공격 성공 / 오답 = 반격당하고 몬스터가 강해짐
4. **포획** — 며칠 간격으로 3연속 정답 시 처치 → 도감 등록
5. **수집** — 도감 완성률·등급별 수집·연속 출석이 장기 동기

## 3. 학습 엔진 (전투 규칙과의 매핑)

| 게임 표현 | 학습 시스템 |
|---|---|
| 몬스터 출현 | 복습 주기가 도래한 단어 (due) |
| 공격 성공 (정답) | 게이지 +1, 다음 출현 1일 뒤 → 3일 뒤 |
| 몬스터 도주 | "잊어버릴 때쯤" 다시 만나는 간격 반복 |
| 반격당함 (오답) | 게이지 초기화, 즉시 재출현 (이번 세션 내 재도전) |
| 몬스터 강화 (등급↑) | 오답 누적 = 취약 단어 → 노출 우선순위 상승 |
| 처치/포획 (게이지 3/3) | 서로 다른 날 3연속 정답 = 장기기억 정착 판정 |

## 4. 전투 규칙 (HP 턴제)

- **용사 HP: 하트 3개.** 던전 입장 시 회복.
- 오답 → 몬스터의 반격, 하트 −1. **하트 0 = 강제 퇴장** (남은 몬스터는 다음 입장 때).
- 정답 → 타격 이펙트 + 몬스터 게이지 −1 (게이지 = 남은 포획 단계 3칸).
- 연속 정답 콤보 카운트, 최고 기록 저장.
- 세션 종료 시 결과창: 처치 수 / 피해 횟수 / 포획 수.

## 5. 등급 시스템

| 등급 | 조건 (누적 오답) | 색 |
|---|---|---|
| COMMON 일반 | 0 | 회색 |
| RARE 희귀 | 1–2 | 파랑 |
| EPIC 영웅 | 3–4 | 보라 |
| LEGENDARY 전설 | 5+ | 금색 |

- 틀릴수록 등급이 오르는 역설 = "나를 가장 많이 이긴 단어"가 도감의 자랑거리.
- 전설 몬스터 처치 시 특별 연출.

## 6. 화면 구성 (5)

1. **마을(홈)** — 도감 통계, 던전 입장 버튼(+오늘 출현 수), 연속 출석
2. **전투** — 던전 배경, 픽셀 몬스터 스프라이트, 금테 파란 대화창, 4지선다 커맨드, 용사 하트·몬스터 게이지
3. **영입소(단어 등록)** — 복붙 등록 (`abandon - 버리다` 등 자동 인식), 백업
4. **도감** — 픽셀 카드 그리드, 등급 필터, 미포획은 실루엣 `???`
5. **결과** — 전투 요약 + 다음 출현 안내

## 7. 비주얼 가이드 (16비트 JRPG)

- **팔레트**: 다크 네이비 던전(#1a1c3a 계열), 대화창 파란 그라데이션 + 금테, HP 빨강/게이지 초록
- **폰트**: 한글 픽셀 폰트 Galmuri11 (웹폰트)
- **몬스터**: 단어별 고정 시드로 절차 생성되는 픽셀 스프라이트 (같은 단어 = 항상 같은 몬스터)
- **연출**: 피격 점멸, 화면 흔들림, 픽셀 파티클, 등장/처치 애니메이션, 8비트 효과음(음소거 가능)
- **image-rendering: pixelated**, 스캔라인은 은은하게(옵션)
